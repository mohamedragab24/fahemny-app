# **App Name**: ConnectNow

## Core Features:

- User Registration and Login: Secure registration and login system for both employers and freelancers.
- Project Posting: Employers can post projects with descriptions, budgets, and deadlines.
- Freelancer Offer Submission: Freelancers can submit proposals for projects, including their skills and proposed rates.
- Simple Chat: Basic real-time chat functionality for communication between employers and freelancers.

## Style Guidelines:

- Primary color: HSL 210, 60%, 50% -> RGB Hex: #3399CC for a professional and trustworthy feel.
- Background color: HSL 210, 20%, 95% -> RGB Hex: #F0F8FF - a light tint of the primary blue to ensure elements 'pop' effectively.
- Accent color: HSL 180, 50%, 40% -> RGB Hex: #33A6A6 to contrast and draw attention to interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text
- Modern and responsive layout using Tailwind CSS grid and flexbox.
- Use a consistent set of line icons from a library like FontAwesome or Feather Icons.
